# 💧 Water Quality Estimator

## 📌 Project Overview

The **Water Quality Estimator** is a Machine Learning-based project that predicts whether water is safe for consumption based on various chemical and physical parameters.
It helps in understanding water safety using data-driven insights.

---

## 🚀 Features

* Predicts water quality (Safe / Unsafe)
* Built using Machine Learning (Random Forest Model)
* User-friendly interface using Python (app.py)
* Handles real-world dataset
* Fast and accurate predictions

---

## 🛠️ Tech Stack

* **Python**
* **Pandas & NumPy** (Data Processing)
* **Scikit-learn** (Model Building)
* **Flask / Streamlit** (if used in app.py)
* **Jupyter Notebook** (Model Training)

## 📊 Dataset Information

The dataset contains water quality parameters such as:

* pH
* Hardness
* Solids
* Chloramines
* Sulfate
* Conductivity
* Organic Carbon
* Trihalomethanes
* Turbidity
